Vitra API Reference

This site is the API reference for Vitra. You can find more detailed documentation, including the whitepaper, specs, and CLI usage, in the git repository.

Quick reference
  • Database. The main API for using Vitra.
    • Database.create(). Create a new Vitra database.
    • Database.load(). Load an existing Vitra database.
    • database#call. Execute a transaction the database.

  • Operation. Individual operations included in a Transaction.
  • OpLog. The API for participant operation logs.
  • IndexLog. The API for the output index log.
